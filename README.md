# 지하철 노선도
## 1단계 - 지하철 정보 관리 기능
### 구현할 기능 목록
- [x] 지하철 노선에 역을 등록 할 수 있다.
- [x] 지하철 노선에 역을 제거 할 수 있다.
- [x] 지하철 노선을 조회한다.
- [x] 지하철 노선 목록들을 조회한다.

## 2단계 - 지하철 정보 관리 기능
### 구현할 기능 목록
- [x] 경로를 조회한다.
  - [x] 출발역과 도착역 사이의 최단 거리 경로를 구하고 최단거리 경로와 총 거리 정보를 함께 응답한다.
  - [x] 경로 조회시 요금 정보를 포함하여 응답한다.
