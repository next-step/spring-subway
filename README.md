### 설계
- 역 간 연결
   - 다음 역
      - Station 객체 
   - 거리
      - Distance 객체
      - 어느 역에 연결되느냐에 따라 다름 (Station 객체로 표현 불가)
   - 방향성
      - 이동 가능한 방향만 그래프에 담기
   > 새 객체 생성 (Section) <br/>
   > 필드 : 다음역(Station), 거리(Distance)

  
- 그래프
  > 객체 명 : SubwayGraph <br/>
  > key : Station (역) <br/>
  > value : List Section (이동 가능 구간) <br/>
